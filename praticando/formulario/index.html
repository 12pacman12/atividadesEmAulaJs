Informações sobre como foi feito este formulário

Para criar o formulário desta página foi utilizado a tag "form", dentro dela foram adicionados labels e inputs para serem nomeados e criados os campos para serem inseridos os dados do formulário.

Propriedades utilizadas

Type

Define o tipo de dado que será inserido no input

Name

Faz uma referencia ao input na hora que ele for enviado

id e for

Nesse caso utilizando formulário, passando o mesmo nome tanto para o "id" quanto para o "for", faz com que a label e o input sejam linkados um ao outro, permitindo ao usuário na hora de selecionar o campo clicar tando no nome da label quanto dentro do input pois os dois estaram unidos.

Value

Para determinados tipos de imputs, é necessário ser passado o tipo de valor que se espera receber, para que na hora que forem enviados possam ser identificados.

Tag button com a propriedade submit

Cria um botão para que possa ser enviado alguma informação.
